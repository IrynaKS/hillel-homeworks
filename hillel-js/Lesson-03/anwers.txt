1. В чём разница между BOM и DOM? 
DOM - объектная модель документа - интерфейс для работы с HTML-документом. С помощью объекта document мы можем обращаться к странице и что-то в ней менять или создавать.
BOM - объектная модель браузера - интерфейс для работы с окном браузера (со всем, кроме документа). 


2. В чём принципиальная разница между выводом сообщений с 
помощью alert и console.log? 
Alert выводит сообщение в виде модального окна в браузер для пользователя. 
console.log выводит сообщения для разработчика в консоль. 


3. Чем отличаются ошибки чтения от ошибок выполнения? 
Ошибки чтения обнаруживаются на этапе считывания кода и выдается ошибка и дальше никаких действий не будет. Ошибки выполнения выявляются уже при выполнении программы (при условии, что нет ошибок чтения (синтаксических ошибок)).


4. Всё ли верно? var m; var M; var 7e;
Не лучшая практика давать имена переменных, состоящие из одной буквы или начинающихся с цифр. 
Оптимальным является использование понятных слов на английском языке. 
Если слов в переменной несколько - лучше использовать camelCase. 
Также названия переменных могут начинаться с нижнего подчеркивания или знака $.


5. Как привести значение любого примитива к булевому? (true/false) 
C помощью функции Boolean(value) либо с помощью оператора логического отрицания !!(value). undefined, null, 0, NaN, пустая строка и false - преобразовываются в false.


6. Какой будет результат выражений? По возможности, объясните почему именно такой 
результат. 
"23" + 20 + '' = ? - 2320 - конкатенация строк (строка + любой другой тип данный = строка)
"1" + (20 + 3) = ? - 123 - строка "сшилась" с выполненной математической операцией в скобках 
(из-за порядка записи операндов) 
"a" / 5 = ? - NaN - т.к. "а" не число, то и результатом деления будет NaN
("str" || 5) > 0 || (10 && true) - true - оператор || предполагает, что хоть одно из выражений будет верным
